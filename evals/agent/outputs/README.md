# Agent Eval Outputs

Saved-output mode can be added here later as:

```text
evals/agent/outputs/<case-id>.md
```

The initial harness only validates deterministic repo contracts. Add saved outputs when there is a real transcript or generated answer worth preserving as a regression fixture.

Before committing a saved output, scrub prompts, private data, tool results, environment-like values, paths that should not be public, screenshots, traces, logs, and any synthetic secret-shaped values that are present only to test redaction.
