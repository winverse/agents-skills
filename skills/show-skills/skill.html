연결 문서

docs/skill-catalog.md는 이 repo의 정적 카탈로그이고, project-snippets/show-skills.md는 프로젝트 연결 문구다. 독립 설치에서는 --root로 skills root를 지정한다.

품질 게이트

node skills/show-skills/scripts/show-skills.ts --compact
node skills/show-skills/scripts/show-skills.ts --root skills --compact
node scripts/validate-skill.ts skills/show-skills
node skills/show-skills/scripts/validate-show-skills.ts skills/show-skills
node scripts/validate-skill-html.ts .
node scripts/validate-skill-repo.ts .

금지와 허용

허용
  • 현재 filesystem에서 스킬을 읽는다.
  • 필요하면 주 스킬과 함께 쓸 스킬을 추천한다.
  • 상세 확인용 파일 경로를 제공한다.
금지
  • 스킬을 자동 설치하지 않는다.
  • 오래된 수동 목록만 믿지 않는다.
  • 목록 요청에서 실제 작업을 실행하지 않는다.
