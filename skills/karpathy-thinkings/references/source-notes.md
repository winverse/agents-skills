# 출처 메모

## public source 기준

이 스킬은 Andrej Karpathy가 공개적으로 말한 coding agent discipline과 community `andrej-karpathy-skills` 계열에서 영감을 받았다. 이 repo는 원문을 그대로 복제하지 않고 local workflow에 맞춘다.

## local adaptation rule 기준

- source wording보다 repo의 실제 실패 모드에 맞춘다.
- 조용한 추측, 과설계, 주변 변경, 검증 부족을 직접 겨냥한다.
- public source가 바뀌면 그대로 베끼지 말고 trigger, workflow, validator boundary만 재평가한다.

## update 시 확인

- `SKILL.md`가 구현 전 사고, 단순성, 외과적 변경, 검증을 유지하는가
- 다른 스킬과 책임이 겹치지 않는가
- HTML guide와 snippet이 같은 메시지를 갖는가
