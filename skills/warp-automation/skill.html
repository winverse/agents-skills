사용 판단 매트릭스

사용

Warp title fallback

Warp UI가 assistant 응답 첫 줄을 title 후보로 잡도록 답변 형식을 맞출 때.

핵심

첫 줄만 선택

사용자 프롬프트를 32자 이내로 정리하고 둘째 줄을 비운 뒤 답변한다.

주의

민감정보 제외

secret, token, private URL, full local path는 title에 넣지 않는다.

건너뜀

자동화 절차

terminal이나 config workflow는 기본 작업이 아니다.

응답 흐름

입출력 스키마

입력

  • 현재 사용자 프롬프트
  • Warp title fallback이 켜져 있다는 사용자 의도
  • 일반 답변을 원하는지 여부

출력

사용자 질문 요약

금지와 허용

허용 금지 별도 요청 필요
첫 줄 title + 빈 둘째 줄 + 셋째 줄 본문 `Q:` 또는 `Prompt:` 접두사 terminal 또는 config workflow
