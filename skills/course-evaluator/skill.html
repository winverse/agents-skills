course-evaluator

학생 답안 CSV, 스프레드시트, 과제 응답, 루브릭을 읽어 학생별 Markdown 평가 파일을 만든다. 점수 산정과 교육적 코멘트를 분리하고, embedded newline을 보존하며, 반복 코멘트를 막는 채점 스킬이다.

사용 판단 매트릭스

사용

학생별 평가 파일

학생 답안을 읽고 평가/학생명.md 형태의 개별 피드백과 점수가 필요할 때.

사용

CSV 구조 보존

답안 cell에 줄바꿈, 쉼표, 따옴표가 있어 line split으로 깨질 위험이 있을 때.

함께 사용

기존 평가 스타일

이전 결과의 형식은 유지하되 문장과 보완 방향은 학생별로 새로 써야 할 때.

건너뜀

요약만 필요한 작업

개별 채점, 루브릭 적용, 개인정보 최소화, 파일 수 검산이 필요 없는 단순 집계.

출력 Markdown 파일 수와 학생 수, 문항별 점수 합계와 총점이 일치해야 한다.

오용 방지

허용
  • RFC 4180 호환 파서나 구조화 CSV 기능을 사용한다.
  • 러프한 채점과 깊은 코멘트를 분리한다.
  • 학생이 맞힌 핵심, 빠진 개념, 고득점 보완 방향을 따로 쓴다.
  • 원본 답안은 필요한 짧은 근거만 인용하거나 요약한다.
금지
  • line split으로 embedded newline 답안을 깨지 않는다.
  • 학생 이름만 바꾼 stock phrase를 반복하지 않는다.
  • 루브릭 문장을 복사해 코멘트를 대신하지 않는다.
  • 개인정보나 원본 CSV 전체를 새 artifact에 불필요하게 붙이지 않는다.
